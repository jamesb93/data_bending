# How to run these scripts | 25/2/20

## What is in here?

Each function is contained to a folder.

The process begins with the mosh executable. This turns data into audio.

The next part of the process begins by performing some kind of descriptor analysis on the audio.

The last part is to use dimension reduction and clustering to create groupings from the audio.

There is an example `bash` script which performs a run of all these parts in sequence.
